# 📞 Phone Region Checker (Desktop Application)

Легковесное и отказоустойчивое десктопное приложение на базе **Electron**, разработанное для автоматизации проверки регионов номеров телефонов из Excel-баз через сторонний API.

Проект решает реальную бизнес-задачу: быстрая пакетная проверка больших массивов телефонных номеров с автоматической записью результатов в исходную таблицу без ручного копирования.

---

## 🚀 Ключевые фичи и архитектурные решения

* **Пакетная асинхронная обработка (Throttling)**: Реализовано разделение запросов на пачки (по 10 номеров в секунду) через асинхронные задержки, что позволяет строго соблюдать лимиты стороннего API и избегать блокировок.
* **Паттерн повторных попыток (Retry Pattern)**: Для повышения отказоустойчивости приложения при нестабильном интернет-соединении внедрен цикл автоповтора запроса (до 3-х раз на один номер) с микропаузами.

* **Продвинутый UX/UI (Интерфейс)**:
  * Встроен интерактивный консольный лог-бокс, показывающий исчерпывающую информацию о статусе работы приложения.
  * Реализована **динамическая строка прогресса** с живым пересчетом процентов и оставшегося времени выполнения.
  * Интеграция с ОС Windows: клик по пути готового файла в интерфейсе открывает Проводник и **автоматически фокусирует нужный Excel-документ**.
  * Индикация на панели задач: иконка приложения окрашивается при изменении статуса проверки, чтобы привлечь внимание пользователя, если окно свернуто.
* **Отказоустойчивость файловой системы**: Приложение проверяет блокировку системных файлов (`EBUSY`) перед чтением и записью. Если пользователь забыл закрыть Excel-таблицу, программа выдаст понятное предупреждение, защищая данные от повреждения.
* **Защита от VPN**: Перед стартом процесса выполняется быстрый тестовый запрос с использованием `AbortController` и таймаутом в 3 секунды. Если у пользователя включен VPN и хост недоступен, приложение прерывает работу на раннем этапе и выводит понятную инструкцию по исправлению.

## 🛠️ Технологический стек

* **Runtime**: Node.js (v20+)
* **Framework**: Electron (v27)
* **Библиотеки**: `xlsx-populate` (высокопроизводительное чтение/запись Excel без запуска самого Excel в фоне)
* **Интерфейс**: Чистый HTML5, CSS3 (Flexbox, CSS-градиенты), Native IPC (Inter-Process Communication).

---

## 📐 Архитектура кода (Декомпозиция)

Проект построен по классическому принципу разделения обязанностей (Single Responsibility):
* **`index.html`** — изолированный фронтенд. Отвечает только за отрисовку, анимацию прогресса и отправку быстрых системных записок (`ipcRenderer.send`/`invoke`).
* **`main.js`** (Main Process) — бэкэнд приложения. Управляет жизненным циклом окон, сетевыми запросами (`fetch`), файловой системой и координирует обработку ошибок через сквозную иерархию `try/catch/throw`.

## 📦 Как запустить проект в режиме разработки

1. Склонируйте репозиторий:
   git clone https://github.com/irin97/phoneChecker
2. Перейдите в папку проекта:
   cd phoneChecker
3. Установите зависимости:
   npm install
4. Запустите приложение:
   npm start

---

## 🔨 Сборка готового приложения (.exe)

Для компиляции исходного кода в независимый исполняемый файл для операционной системы Windows используется настроенный сборщик `electron-packager`.

Запуск сборки:
npm run build

После успешного завершения готовый продукт будет находиться в директории `dist/PhoneRegionChecker-win32-x64`, откуда его можно упаковать в ZIP-архив и передать конечному пользователю.
